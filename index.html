Web Developer

Jul 2017 - Sept 2018

At GEMS Education I developed booksforacause.me independently. The website enables students of GEMS Modern Academy to trade used textbooks and the proceeds are donated to charity. I pitched it to the Senior Leadership Team to acquire funding, spearheaded a successful marketing campaign, and gained more than 450 users in the first 2 months. I coded everything in HTML, CSS, JavaScript, JQuery, PHP, and MySQL. After I graduated, I trained a team of 4 students to maintain the website and add new features. I'm amazed at how they've managed to keep gaining users, which I would argue is the hardest part.

Software Engineer Intern

Summer 2017

At ESRI I developed plugins to provide support for WEBP and PDF file formats in ArcGIS (ESRI's flagship software) using the open source library GDAL. I then had to write user documentation for the installation process of these plugins. Finally, I collaborated with a team of interns to write and optimize a script which checks for read and write support for various file formats and compares image statistics when converting from one file format to another. Initially this painstaking process had to be performed manaually but our automation script shaved hours off the development cycle. I learned about Makefiles and used C++, Python, and Bash to perform these tasks.
